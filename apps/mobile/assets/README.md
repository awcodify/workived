# Assets

Placeholder for app assets. You'll need to add:

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `splash.png` - Splash screen image (2048x2048)
- `favicon.png` - Web favicon (48x48)

For now, Expo will use default placeholder assets.

## Generating Assets

Use https://www.appicon.co/ or similar tools to generate all required sizes.
