Import Tasks from Linear

Migrate all tasks from Linear to Workived including markdown, images, and comments

Quick Start

  1. Get your Linear API key from Linear Settings → API
  2. Select the target organization and enter the Linear API key
  3. Click "1. Load Task Lists" → "2. Load States" → Click "Smart map by type" for intelligent auto-mapping
  4. Optionally: Click "1. Load Employees" → "2. Load Users" → Auto-matches by email instantly!
  5. Review mappings and use Dry Run to preview before importing

Required Settings

API key is auto-loaded from config. Enter a new one to override.

💡 Tip: Either provide this OR use the "Load Users from Linear" button below to map each Linear user to a Workived employee

Import Mapping

How Linear tasks will be mapped to Workived

Linear Task Status → Workived Task List

Tasks will be automatically sorted into different task lists based on their Linear workflow state (e.g., "Backlog" → List A, "In Progress" → List B, "Done" → List C).

Linear States → Workived Completion Status

Completed/Canceled tasks in Linear will be marked as complete in Workived. Others will remain active.

Linear Priority → Workived Priority

Urgent→Urgent, High→High, Normal→Medium, Low→Low

What Will Be Imported

Task Details: Title, description, priority, due date, assignee
Markdown Content: All formatting preserved (bold, italic, code blocks, lists)
Comments: All comments with markdown and timestamps
Images: Downloaded from Linear and uploaded to Workived storage
